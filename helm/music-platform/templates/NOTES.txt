1. Get API service details:
   kubectl get svc {{ include "music-platform.apiServiceName" . }} -n {{ .Release.Namespace }}

2. Access the API locally with port-forward:
   kubectl port-forward svc/{{ include "music-platform.apiServiceName" . }} 8000:{{ .Values.api.service.port }} -n {{ .Release.Namespace }}

3. Open API docs:
   http://127.0.0.1:8000/docs
